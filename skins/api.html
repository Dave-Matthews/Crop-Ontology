API

This is the official API for the Ontology Curation Tool. It allows you to programmatically retrieve and interact with Ontology data.

To let us gather feedback you can leave a comment using the form on the right sidebar.

API Data Types

Data can be requested in JSON or JSONP formats. The default is JSON. To ask for another type, append the format to the querystring. Example:

http://example.com/page
http://example.com/page?format=jsonp&callback=yourFunction
